# JavaScript-sort
Add different sorting capabilities to native Array

# Example
var arr = [0,1,2,7,6,3,5,4];

### Bubble Sort
arr.bubbleSort();    // [0,1,2,3,4,5,7]

### Selection Sort
arr.selectionSort(); // [0,1,2,3,4,5,7]

### Insertion Sort
arr.insertionSort(); // [0,1,2,3,4,5,7]

### Merge Sort
arr.mergeSort();     // [0,1,2,3,4,5,7]
